Этот плагин предназначен для автоматического зачисления пользователей в глобальные группы (cohort, далее ГГ).
Для шаблона имени ГГ используются значения полей профиля и любые дрегие символы.
Список всех доступных полей вы можете узнать, скачав пользователей (Массовые действия с пользователями -> скачать).

Используются 3 шаблона:
1. Шаблон названия группы - сюда можно написать названия полей, из которых будет стротится название ГГ. Перед названием поля нужно поставить %, каждый новый шаблон - с новой строки.
НАПРИМЕР: %institution или %profile_field_что_нибудь
2. Заменитель для пустых значений - используется для замены пустых значений полей
НАПРИМЕР: Не указано
3. Заменители текста - В получившемся из п.п. 1-2 значении можно заменить одно слово на другое. Каждая замена с новой строки. Формат - старое значение|новое значение
НАПРИМЕР: Не указано|1

Зачисление происходит, когда пользователь входит в систему. Если соответствующая ГГ не найдена, то она создается.
